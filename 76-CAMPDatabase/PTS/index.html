Version History

v3.1.0

  • Added "Detailed View" pages for each item, that lists additional info compared to what's on the main page cards
  • Various minor UI improvements
  • Changed how budget tier cutoff points are defined. Notably, only items with <1 FU budget cost are now considered Very Low cost, 1 FU cost items are now considered Low cost, and 2 FU items are considered Medium cost
  • "Search by: Challenge/Plan name" has been split up into its 2 respective categories
  • Fixed an edge case of item budget being underreported affecting a handful of lights
  • Added subcategory filters

v3.0.3

  • Fixed certain plushies unintentionally being labeled as Meat Week rewards

v3.0.2

  • Fixed budget values being underreported, primarily affecting lights and prefabs
  • Whichever page is the current page is now better shown to the user

v3.0.1

  • Added technical info to each entry
  • known issue: certain budget values are being underreported

v3.0

  • Added CAMP budget cost to items. Massive thank you to Eckserah for helping out with the script side of this.
  • Added support for items unlocked by challenges
  • Added support for items that have "loot to unlock" placement conditions
  • Fixed various items and/or placement conditions being missing
  • Added more placement condition icons
  • Added "sort by budget"
  • Various UI fixes
  • Added Info & FAQ section

v3.0 Beta

  • Expansion towards a more generalized database tool
  • Added category icons
  • Added placement conditions (plan, entitlement) for objects, and added respective source icons, i.e. SCORE, Gold Bullion, Stamps, Atomic Shop icons. Known issue with beta version: certain plans will not show yet (i.e. Nuclear Winter stash boxes), nor do challenge requirements (i.e. Kelp Curtain).
